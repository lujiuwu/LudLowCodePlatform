#### Lud低代码开发平台是什么？

1. Lud低代码开发平台是基于Vue3.0开发、集成Element-Plus组件库，通过拖拽快速生成页面的平台
2. 类似 [易企秀](http://www.eqxiu.com/)、[Maka](http://maka.im/)、[百度 H5](https://h5.baidu.com) 等平台
